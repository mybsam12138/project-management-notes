# Jira Overview

## 1. Definition

Jira is a project management and issue tracking tool used to plan,
track, and manage software development work.

------------------------------------------------------------------------

## 2. Core Concepts

### 2.1 Issue

The basic unit of work in Jira.

Types include: - Bug - Task - Story - Epic

------------------------------------------------------------------------

### 2.2 Workflow

Represents the lifecycle of an issue.

Typical flow:

    To Do → In Progress → Review → Done

------------------------------------------------------------------------

### 2.3 Board

Visual representation of work:

-   Kanban board
-   Scrum board

------------------------------------------------------------------------

### 2.4 Sprint

A time-boxed iteration (e.g., 2 weeks) used in agile development.

------------------------------------------------------------------------

## 3. Key Features

-   Task and bug tracking
-   Workflow management
-   Agile support (Scrum / Kanban)
-   Collaboration (comments, attachments)
-   Reporting and dashboards

------------------------------------------------------------------------

## 4. Typical Workflow

    Product Manager creates tasks
            ↓
    Developers pick tasks
            ↓
    Implement features / fix bugs
            ↓
    Update issue status
            ↓
    Complete and close tasks

------------------------------------------------------------------------

## 5. Integration with Development Tools

Jira is commonly used together with:

    Jira → task management
    GitLab / GitHub → code management
    CI/CD → build and deployment

------------------------------------------------------------------------

## 6. Example

    Issue: Implement audit logging
    Type: Story
    Status: In Progress
    Assignee: Developer

------------------------------------------------------------------------

## 7. Summary

Jira is a centralized system for managing tasks, tracking progress, and
supporting collaboration in software development projects.
