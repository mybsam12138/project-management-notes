# ADR-001: Use RESTful API

## Status
Accepted

## Context
Frontend (Vue 3) communicates with backend (Spring Boot). Need simple and scalable API.

## Decision
Use RESTful API.

## Alternatives
- GraphQL
- gRPC

## Reason
- Simplicity
- Team familiarity
- Strong ecosystem

## Trade-offs
- Less flexible than GraphQL
